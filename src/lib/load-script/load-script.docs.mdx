It adds script tag to the page, waits until the script will be executed and resolves returned promise.
It doesn't add the same script second time, and instead returns the promise of the first call.

## Usage

```js
import loadScript from './load-script'

loadScript('//site.com/script.js')
  .then(
    () => console.log('script loaded'),
    () => console.log('script failed to load'),
  )
```
