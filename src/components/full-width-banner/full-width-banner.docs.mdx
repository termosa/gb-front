### A FullWidthBanner component that renders banner with given background image and text

You can customize FullWidthBanner by setting text, text color, text position and background image:

```jsx
<FullWidthBanner
  text="It’s more than amazing scents and stunning colors — essential oils and natural minerals  will leave your skin feeling so soft, you’ll be feel like silk."
  title="Bath Bombs"
  imageUrl="https://lorempixel.com/1440/700/abstract/7/"
  textColor="#fff"
  position="left"
/>
```
